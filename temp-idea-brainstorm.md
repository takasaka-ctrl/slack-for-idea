# 新しいアイデア創出セッション
Date: 2026-02-24
Purpose: 朝のアイデア創出 cron job

## ブレインストーミング

### 候補アイデア

1. **Smart Screenshot Manager**
   - 問題: スクショが増えすぎて探せない、整理されない
   - 解決: AI-OCR + 自動分類 + 検索可能化
   - ターゲット: デジタルワーカー、デザイナー、エンジニア

2. **Meeting ROI Tracker**
   - 問題: 会議の費用対効果が不明
   - 解決: 自動コスト計算 + 成果トラッキング
   - ターゲット: マネージャー、経営層

3. **Personal API**
   - 問題: 個人データが分散して統合困難
   - 解決: 統一APIで自分のデータにアクセス
   - ターゲット: テックサビー、自動化好き

4. **Code Snippet Library Pro**
   - 問題: 過去に書いたコードを探せない
   - 解決: AI検索 + コンテキスト理解 + 自動タグ付け
   - ターゲット: エンジニア

5. **Time Block Optimizer**
   - 問題: 時間管理が難しい、タスクが溢れる
   - 解決: AIが最適な時間配分を提案、自動調整
   - ターゲット: 忙しいプロフェッショナル全般

## 選定: Time Block Optimizer

**選定理由:**
- 広いターゲット層（ビジネスパーソン全般）
- 明確なペインポイント
- 既存ツール（カレンダー、TODO）との連携可能
- プロトタイプ作成が比較的容易
- 視覚的なUIで差別化しやすい

## 詳細展開

### コンセプト
「AIが勝手に理想のスケジュールを作ってくれるタイムブロッキングアプリ」

### コア機能
1. タスクリスト読み込み（ToDoist, Notion, 手動入力）
2. 各タスクの所要時間をAI推定
3. カレンダーの空き時間を分析
4. 最適な時間配分を自動生成
5. ドラッグ&ドロップで微調整可能
6. 実績トラッキングで精度向上

### 差別化ポイント
- 単なるカレンダーではなく「最適化エンジン」
- タスクの重要度・緊急度・エネルギーレベルを考慮
- 「理想的な1日」をワンクリックで生成
