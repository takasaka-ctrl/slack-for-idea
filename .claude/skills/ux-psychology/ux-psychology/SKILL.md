# UX Psychology

このスキルは、ユーザーエクスペリエンス設計に必要な心理学的原則とデザインパターンを提供します。

## Overview

UXデザインにおいて、ユーザーの意思決定や行動を理解することは不可欠です。このスキルでは以下のカテゴリーをカバーします：

- 認知と知覚の原則
- バイアスと意思決定
- フレーミングと提示方法
- 損失回避と所有効果
- 社会的影響
- モチベーションと習慣形成
- 体験の記憶
- パフォーマンスと速度の最適化

## 基本的な使用方法

### 1. 適切な心理原則の選択

デザイン課題に応じて、関連する心理原則を選択します：

**情報設計の場合:**
- 認知負荷を減らすために、段階的開示や視覚的階層を活用
- 選択的注意を考慮した情報の優先順位付け
- 系列位置効果を活用したナビゲーション設計

**意思決定の促進:**
- アンカー効果やおとり効果を使った価格設計
- デフォルト効果を活用したオプション設定
- 決断疲れを避けるための選択肢の制限

**信頼性の構築:**
- 社会的証明でユーザーレビューや使用統計を表示
- ハロー効果を考慮したブランド体験の一貫性
- 美的ユーザビリティ効果による視覚的品質の向上

**エンゲージメントの向上:**
- ゲーミフィケーションとプログレス表示
- 変動型報酬によるリテンション向上
- ツァイガルニク効果を活用した継続的関与

### 3. 複数の原則の組み合わせ

効果的なUXデザインでは、複数の原則を組み合わせます：

**例：オンボーディングフロー**
1. 段階的要請（小さなステップから）
2. プログレス表示（目標勾配効果）
3. 社会的証明（他のユーザーの成功事例）
4. ピーク・エンドの法則（印象的な終了体験）

**例：価格ページ**
1. アンカー効果（高額プランを先に表示）
2. おとり効果（中間プランを魅力的に）
3. 希少性効果（限定オファー）
4. 社会的証明（人気プランの表示）

### 4. ユーザー調査での活用

ユーザーテストやアンケート設計時の注意点：
- 確証バイアスを避ける多角的な質問設計
- 観察効果を考慮した自然な行動の観察
- 調査バイアスを最小化するサンプリング
- 期待バイアスを排除した中立的な質問

## 設計プロセスでの統合

### 1. 問題の分析
ユーザーが直面している課題を心理的視点から分析：
- どの認知プロセスが関与しているか
- どのバイアスが影響しているか
- どの動機づけ要因が重要か

### 2. 仮説の立案
心理原則に基づいた設計仮説を立てる：
「〇〇の原則を適用すれば、〇〇の行動が促進されるはず」

### 3. プロトタイピング
心理原則を具体的なデザインパターンに変換

### 4. 検証
A/Bテストやユーザーテストで仮説を検証

### 5. 反復
データに基づいて改善を繰り返す

## よくある適用シナリオ

### コンバージョン率の向上
- 損失回避：「今だけの特典を逃さないで」
- 社会的証明：レビューと評価の表示
- 希少性：在庫の残数表示
- アンカー効果：価格の戦略的配置

### ユーザーエンゲージメントの向上
- ゲーミフィケーション：ポイントとバッジ
- 目標勾配効果：プログレス表示
- 変動型報酬：予測不可能な報酬
- ツァイガルニク効果：未完了タスクのリマインダー

### ユーザビリティの改善
- 認知負荷の削減：情報の整理と階層化
- 親近性バイアス：既存パターンの活用
- 段階的開示：必要な時に必要な情報
- 視覚的階層：重要度に応じた配置

### 信頼性の構築
- 社会的証明：ユーザー数や事例
- ハロー効果：一貫したブランド体験
- 美的ユーザビリティ：高品質なビジュアル
- 透明性：プロセスの可視化

## 倫理的配慮

心理原則を適用する際は、ユーザーの利益を最優先に：

1. **透明性**: ユーザーに意思決定の情報を提供
2. **選択の自由**: 強制せず、選択肢を保持
3. **誠実さ**: ダークパターンを避ける
4. **公平性**: すべてのユーザーに配慮
5. **プライバシー**: データの適切な扱い
