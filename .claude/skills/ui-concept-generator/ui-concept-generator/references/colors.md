## 避けるべきパターン

以下の組み合わせはAI感が強いため避ける：

- 白背景 + 紫〜青グラデーション (#8b5cf6 → #3b82f6)
- デフォルトのblue-500単色アクセント
- gray-100背景 + gray-900テキストのみ
- 彩度100%の原色のみの使用
