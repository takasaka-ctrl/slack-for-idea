## Web Pages

### landing-page
- **説明**: 製品/サービス紹介、コンバージョン獲得
- **推奨レイアウト**: Hero → Features → Social Proof → CTA
- **推奨スタイル**: Bold, High-impact
- **情報密度**: medium (3)
- **主要コンポーネント**: Hero, Feature cards, Testimonials, Pricing, CTA button

### corporate-website
- **説明**: 企業情報、会社案内
- **推奨レイアウト**: Multi-page with consistent header/footer
- **推奨スタイル**: Corporate Classic, Swiss Minimal
- **情報密度**: medium (3)
- **主要コンポーネント**: Navigation, Hero, About, Services, Contact

### portfolio
- **説明**: 個人/企業の作品集
- **推奨レイアウト**: Grid-based project showcase
- **推奨スタイル**: Minimal, Editorial
- **情報密度**: low-medium (2-3)
- **主要コンポーネント**: Project grid, Project detail, About, Contact

### blog
- **説明**: 記事・コンテンツ配信
- **推奨レイアウト**: Article list + Detail view
- **推奨スタイル**: Editorial, Minimal
- **情報密度**: medium-high (3-4)
- **主要コンポーネント**: Article cards, Article body, Author bio, Related posts

### documentation
- **説明**: 技術ドキュメント、ヘルプセンター
- **推奨レイアウト**: Sidebar navigation + Content area
- **推奨スタイル**: Clean, Functional
- **情報密度**: high (4)
- **主要コンポーネント**: Sidebar nav, Search, Code blocks, TOC, Breadcrumbs

### pricing-page
- **説明**: 料金プラン比較
- **推奨レイアウト**: Tier comparison cards
- **推奨スタイル**: Clear, Trust-building
- **情報密度**: medium (3)
- **主要コンポーネント**: Pricing cards, Feature comparison, FAQ, CTA

## Applications

### dashboard
- **説明**: データ可視化、KPIモニタリング
- **推奨レイアウト**: Sidebar + Header + Main content
- **推奨スタイル**: Dark Mode, Minimal, Bento Grid
- **情報密度**: high (4-5)
- **主要コンポーネント**: KPI cards, Charts, Tables, Activity feed, Filters

### admin-panel
- **説明**: 管理画面、バックオフィス
- **推奨レイアウト**: Sidebar nav + Data tables
- **推奨スタイル**: Functional, Clean
- **情報密度**: high (4-5)
- **主要コンポーネント**: Data tables, Forms, CRUD actions, Bulk operations

### saas-app
- **説明**: SaaSプロダクトのメイン画面
- **推奨レイアウト**: App shell with workspace
- **推奨スタイル**: Modern, Productive
- **情報密度**: medium-high (3-4)
- **主要コンポーネント**: Command palette, Lists, Detail panels, Settings

### crm
- **説明**: 顧客管理システム
- **推奨レイアウト**: List → Detail view, Kanban
- **推奨スタイル**: Professional, Data-rich
- **情報密度**: high (4-5)
- **主要コンポーネント**: Contact cards, Pipeline view, Activity timeline

### project-management
- **説明**: タスク・プロジェクト管理
- **推奨レイアウト**: Kanban, List, Calendar views
- **推奨スタイル**: Clean, Colorful status
- **情報密度**: medium-high (3-4)
- **主要コンポーネント**: Task cards, Kanban board, Gantt chart, Filters

### analytics
- **説明**: データ分析、レポート
- **推奨レイアウト**: Chart-heavy dashboard
- **推奨スタイル**: Data-focused, Dark mode
- **情報密度**: high (5)
- **主要コンポーネント**: Complex charts, Data tables, Export, Date range picker

## Mobile

### mobile-app
- **説明**: 一般的なモバイルアプリ
- **推奨レイアウト**: Bottom nav + Content screens
- **推奨スタイル**: Platform-native feel
- **情報密度**: medium (3)
- **主要コンポーネント**: Bottom navigation, Cards, Lists, FAB

### mobile-ecommerce
- **説明**: モバイルEC/ショッピングアプリ
- **推奨レイアウト**: Browse → PDP → Cart flow
- **推奨スタイル**: Visual, Product-focused
- **情報密度**: medium-high (3-4)
- **主要コンポーネント**: Product grid, Product detail, Cart, Checkout

### mobile-fintech
- **説明**: 金融・決済アプリ
- **推奨レイアウト**: Home → Transactions → Actions
- **推奨スタイル**: Trust-building, Clean
- **情報密度**: medium (3)
- **主要コンポーネント**: Balance card, Transaction list, Quick actions

### mobile-social
- **説明**: SNS・コミュニケーションアプリ
- **推奨レイアウト**: Feed-based, Tab navigation
- **推奨スタイル**: Engaging, Content-first
- **情報密度**: medium (3)
- **主要コンポーネント**: Feed, Stories, Composer, Profile

### mobile-onboarding
- **説明**: アプリ初回起動時のオンボーディング
- **推奨レイアウト**: Multi-step carousel
- **推奨スタイル**: Welcoming, Illustrated
- **情報密度**: low (2)
- **主要コンポーネント**: Illustrations, Progress dots, Skip/Next buttons

## Components

### form
- **説明**: 入力フォーム（登録、問い合わせ等）
- **推奨レイアウト**: Single column, Progressive
- **推奨スタイル**: Clean, Accessible
- **情報密度**: medium (3)
- **主要コンポーネント**: Input fields, Validation, Submit button

### modal
- **説明**: モーダルダイアログ
- **推奨レイアウト**: Centered overlay
- **推奨スタイル**: Focused, Minimal
- **情報密度**: low (2)
- **主要コンポーネント**: Header, Content, Actions

### card
- **説明**: 情報カードコンポーネント
- **推奨レイアウト**: Flexible content container
- **推奨スタイル**: Varies by context
- **情報密度**: medium (3)
- **主要コンポーネント**: Image, Title, Description, Actions

### navigation
- **説明**: ナビゲーションコンポーネント
- **推奨レイアウト**: Horizontal or Vertical
- **推奨スタイル**: Clear, Consistent
- **情報密度**: low-medium (2-3)
- **主要コンポーネント**: Nav items, Dropdown, Mobile menu

### data-table
- **説明**: データテーブル
- **推奨レイアウト**: Rows with sortable columns
- **推奨スタイル**: Functional, Dense
- **情報密度**: high (5)
- **主要コンポーネント**: Headers, Rows, Pagination, Search, Filters

### chart
- **説明**: チャート・グラフコンポーネント
- **推奨レイアウト**: Chart + Legend + Controls
- **推奨スタイル**: Data-focused
- **情報密度**: high (4-5)
- **主要コンポーネント**: Chart area, Axis, Legend, Tooltip

### hero
- **説明**: ヒーローセクション
- **推奨レイアウト**: Full-width, Above fold
- **推奨スタイル**: High-impact, Bold
- **情報密度**: low (2)
- **主要コンポーネント**: Headline, Subheadline, CTA, Visual

### pricing-card
- **説明**: 料金プランカード
- **推奨レイアウト**: Vertical feature list
- **推奨スタイル**: Comparison-friendly
- **情報密度**: medium (3)
- **主要コンポーネント**: Plan name, Price, Features, CTA

### testimonial
- **説明**: お客様の声
- **推奨レイアウト**: Quote + Attribution
- **推奨スタイル**: Trust-building
- **情報密度**: low (2)
- **主要コンポーネント**: Quote, Avatar, Name, Title

### feature-section
- **説明**: 機能紹介セクション
- **推奨レイアウト**: Grid or Alternating
- **推奨スタイル**: Visual, Scannable
- **情報密度**: medium (3)
- **主要コンポーネント**: Icon/Image, Title, Description

## 業界別推奨設定

### Tech / SaaS
- **スタイル**: Modern Minimal, Dark Mode, Bento Grid
- **カラー**: Slate, Linear Inspired, Vercel Mono
- **フォント**: Space Grotesk, DM Sans

### Healthcare
- **スタイル**: Clean, Accessible, Trust-building
- **カラー**: Medical Trust, Wellness Green
- **フォント**: Manrope, Source Sans 3

### Finance
- **スタイル**: Corporate Classic, Professional
- **カラー**: Bank Navy, Investment Green
- **フォント**: DM Serif Display, DM Sans

### E-commerce
- **スタイル**: Product-focused, Visual
- **カラー**: Industry-specific (Shopify, Luxury)
- **フォント**: Plus Jakarta Sans, Outfit

### Creative / Agency
- **スタイル**: Bold, Expressive, Neo Brutalism
- **カラー**: Gradient Vibrant, Bold Agency
- **フォント**: Clash Display, Syne

### Japanese Market
- **スタイル**: Information-dense, Trust-focused
- **カラー**: Japanese Corporate Blue, PayPay Red
- **フォント**: Noto Sans JP, BIZ UDPGothic
