# Japan Market UI Reference

日本市場向けUI設計の専門ガイド。文化的・言語的特性を考慮した設計ルール。

---

## 概要

日本市場向けUIは、欧米市場とは異なる特性を持つ：
- **情報密度**: 高めが好まれる（「余白は無駄」という感覚）
- **信頼構築**: 実績・資格・メディア掲載が重要
- **詳細志向**: FAQや注意書きを隠さない
- **形式重視**: 敬語、会社概要、電話番号

## タイポグラフィ

### 基本設定

```css
/* 日本語本文 */
body {
  font-family: "Noto Sans JP", "Hiragino Sans", "Hiragino Kaku Gothic ProN", 
               "Yu Gothic", "Meiryo", sans-serif;
  font-size: clamp(0.9375rem, 1.2vw + 0.5rem, 1rem); /* 15-16px */
  line-height: 1.8;  /* 欧文の1.5より広く */
  letter-spacing: -0.02em;  /* 欧文より詰める */
  font-feature-settings: "palt" 1;  /* プロポーショナル */
}

/* 日本語見出し */
h1, h2, h3 {
  font-weight: 700;
  line-height: 1.4;  /* 本文より狭く */
  letter-spacing: 0;  /* 見出しは詰めない */
}
```

### 行間・字間ルール

| 要素 | 欧文 | 日本語 | 理由 |
|------|------|--------|------|
| 本文行間 | 1.5 | 1.8-2.0 | 漢字の複雑さで視認性確保 |
| 本文字間 | 0 | -0.02em | 和文は詰めた方が自然 |
| 見出し行間 | 1.2 | 1.3-1.4 | 長い日本語見出し対応 |
| 見出し字間 | 0.05em | 0 | 和文見出しはベタ組み |

### フォント選択基準

**ゴシック体（sans-serif相当）**
- `Noto Sans JP`: 汎用性高、Google Fonts
- `Hiragino Sans`: Apple環境で美しい
- `BIZ UDPGothic`: アクセシビリティ重視
- `Zen Kaku Gothic New`: モダン、スタートアップ向け
- `M PLUS 1p`: 丸みがある、カジュアル

**明朝体（serif相当）**
- `Noto Serif JP`: 汎用フォーマル
- `Shippori Mincho`: 上品、伝統的
- `Zen Old Mincho`: クラシック

**日本語＋欧文の組み合わせ**
```css
/* 推奨パターン */
font-family: "Space Grotesk", "Noto Sans JP", sans-serif;
font-family: "DM Sans", "Zen Kaku Gothic New", sans-serif;

/* 数字・英語の見やすさを優先 */
.numbers {
  font-family: "Roboto Mono", "Noto Sans JP", sans-serif;
  font-variant-numeric: tabular-nums;
}
```

### 句読点処理

```css
/* 句読点ぶら下がり */
p {
  hanging-punctuation: allow-end;
}

/* 禁則処理 */
p {
  word-break: normal;
  overflow-wrap: anywhere;
  line-break: strict;  /* 厳格な禁則 */
}
```

## 情報密度

### 日本市場の期待値

欧米LPが「余白で高級感」を演出するのに対し、日本市場では：
- **情報が多い = 誠実・詳しい**
- **余白が多い = 手抜き・不安**

### 必須要素チェックリスト

**信頼バッジ（ファーストビュー付近に配置）**
- [ ] 導入実績数（「○○社導入」）
- [ ] メディア掲載（「○○で紹介されました」）
- [ ] 受賞歴
- [ ] 資格・認証マーク
- [ ] お客様ロゴ一覧

**詳細情報（折りたたみでも全部見せる）**
- [ ] FAQ（10問以上でも可）
- [ ] 料金の注意書き
- [ ] 利用規約・プライバシーポリシーへのリンク
- [ ] 会社概要リンク

**連絡手段（目立つ位置に）**
- [ ] 電話番号（ヘッダーに大きく）
- [ ] お問い合わせボタン
- [ ] 営業時間

### 情報密度の調整方法

```
低密度（欧米スタイル）→ 日本市場向け調整

Before:
┌─────────────────────────────────┐
│                                 │
│         Big Hero Text           │
│                                 │
│          [ CTA ]                │
│                                 │
└─────────────────────────────────┘

After:
┌─────────────────────────────────┐
│ ☎ 0120-XXX-XXX  [お問い合わせ]   │
├─────────────────────────────────┤
│  Hero Text     │ 🏆 ○○賞受賞    │
│                │ 📺 TVで紹介     │
│  [ CTA ]       │ ✓ 導入3000社   │
└─────────────────────────────────┘
```

```

│  Tel: 0120-XXX-XXX  [お問い合わせ]    │ ← 電話番号必須
├─────────────────────────────────────┤

│  [サービス] [料金] [事例] [会社概要]   │
├─────────────────────────────────────┤

│   「○○業界シェアNo.1」              │

├─────────────────────────────────────┤
│   サービス特徴（3-4つ）              │
├─────────────────────────────────────┤
│   導入事例（詳細に）                 │
│   会社名・担当者名・顔写真            │
├─────────────────────────────────────┤
│   料金プラン + 比較表                │
├─────────────────────────────────────┤
│   FAQ（折りたたみ、10問以上可）       │

│   または [資料請求] [デモ予約]        │
├─────────────────────────────────────┤

│   所在地、設立、資本金、代表者        │
```

### 日本企業Webサイトの特徴

**ヘッダー**
- 電話番号を目立つ位置に
- 「お問い合わせ」ボタン必須
- ロゴは左、連絡手段は右

**フッター**
- 会社概要を詳しく
- プライバシーポリシー、利用規約
- サイトマップリンク
- SNSリンク

## カラー

### 日本市場で信頼されやすい色

| 色 | 用途 | 心理効果 |
|----|------|----------|
| 紺（#003c71）| 大企業、金融、官公庁 | 信頼、誠実、安定 |
| 青（#0066cc）| IT、サービス | 清潔、知性、冷静 |
| 緑（#00875a）| ヘルスケア、エコ | 安心、自然、成長 |
| 赤（#c41e3a）| 伝統、食品、セール | 情熱、活力、注目 |
| 金（#c9a227）| 高級、伝統工芸 | 価値、品格 |

### 避けるべき色使い

- **原色のみ**: 子供っぽく見える
- **紫単色**: 日本ではスピリチュアル・葬儀の印象
- **黒背景 + 白文字のみ**: 威圧的（アクセント必要）

### 日本企業でよく使われるパレット

**Corporate Blue（信頼系）**
```css
--primary: #003c71;
--secondary: #5a5a5a;
--accent: #0066cc;
--background: #f8f8f8;
--text: #333333;
```

**Warm Traditional（和モダン）**
```css
--primary: #1a1a1a;    /* 墨色 */
--secondary: #4a4a4a;
--accent: #8b4513;     /* 茶 */
--background: #f5f5f0; /* 生成り */
--text: #333333;
```

## フォーム設計

### 日本市場向けフォームの特徴

**必須項目を明示**
```html
<label>
  会社名 <span class="required">必須</span>
</label>
```

**ふりがな入力**
- 名前入力にはふりがな欄が必要
- 自動入力（IME連携）があると便利

**住所入力**
- 郵便番号から住所自動入力
- 都道府県 → 市区町村 → 番地の順

**電話番号**
- ハイフンあり/なし両対応
- 携帯・固定の自動判別

### プレースホルダー例

```html
<input placeholder="例）株式会社○○" />
<input placeholder="例）山田 太郎" />
<input placeholder="例）ヤマダ タロウ" />
<input placeholder="例）000-0000" />
<input placeholder="例）03-0000-0000" />
<input placeholder="例）example@company.co.jp" />
```

## コピーライティング

### 敬語レベル

| 場面 | 敬語レベル | 例 |
|------|-----------|-----|
| CTA | です・ます調 | 「今すぐ資料請求する」 |
| 説明文 | です・ます調 | 「○○を実現します」 |
| エラー | 丁寧語 | 「入力内容をご確認ください」 |
| フッター | です・ます調 | 「お気軽にお問い合わせください」 |

### 避けるべきフレーズ

**欧米的すぎる表現**
- ❌ "Get Started" → ⭕ "無料で始める" / "今すぐ登録"
- ❌ "Learn More" → ⭕ "詳しく見る" / "もっと詳しく"
- ❌ "Contact Us" → ⭕ "お問い合わせ"
- ❌ "Subscribe" → ⭕ "メルマガ登録" / "購読する"

**AI翻訳っぽい表現**
- ❌ 「あなたの」の多用
- ❌ 「私たちは」の多用
- ❌ 不自然に長い主語

## 縦書き対応（和風サイト向け）

### CSSでの縦書き

```css
.vertical-text {
  writing-mode: vertical-rl;
  text-orientation: mixed;
  /* 英数字は横向きに */
}

.vertical-text .number {
  text-combine-upright: digits 2;
  /* 2桁までの数字を横並びに */
}
```

### 縦書きが適切な場面

- 和食店、旅館のサイト
- 伝統工芸、書道関連
- 和風ブランディング
- アクセント的な使用

## レスポンシブ考慮

### 日本市場のデバイス特性

- **iPhone比率が高い**: iOS最適化重要
- **高齢者層**: 大きめのタップターゲット
- **通勤時間の利用**: 片手操作対応

### フォントサイズ推奨

```css
/* モバイル（日本語） */
@media (max-width: 640px) {
  body {
    font-size: 15px;  /* 欧文より大きめ */
    line-height: 1.9;
  }
  
  h1 {
    font-size: 24px;  /* 長い日本語見出し対応 */
  }
}
```
