## Modern Sans Pairs

### Space Grotesk + DM Sans
- **Heading**: Space Grotesk (500, 700)
- **Body**: DM Sans (400, 500)
- **Import**: `https://fonts.googleapis.com/css2?family=Space+Grotesk:wght@500;700&family=DM+Sans:wght@400;500&display=swap`
- **特徴**: テック感、モダン、読みやすい
- **適用**: SaaS, スタートアップ, 開発者ツール

### Satoshi + Inter
- **Heading**: Satoshi (700, 900)
- **Body**: Inter (400, 500)
- **Import**: Satoshiは@font-faceで読み込み、Inter: `https://fonts.googleapis.com/css2?family=Inter:wght@400;500&display=swap`
- **特徴**: クリーン、プロフェッショナル
- **適用**: プロダクト、コーポレート
- **注意**: Interは本文のみ、見出しには使わない

### 日本語 + 欧文の組み合わせ推奨

日本語フォントと欧文フォントを組み合わせる場合：
- **数字**: Roboto Mono, Tabular Lining
- **英語見出し**: Space Grotesk, Clash Display
- **英語本文**: DM Sans, Source Sans 3

```css
font-family: "Space Grotesk", "Noto Sans JP", sans-serif;
```

## 避けるべき組み合わせ

以下はAI感が強くなるため避ける：

- Inter + Inter（見出し・本文同じ）
- Roboto + Roboto
- Arial + Arial
- Open Sans + Open Sans（単調）
- system-ui単独使用

## フォントサイズ推奨

```css
/* 見出し */
h1: clamp(2.5rem, 5vw, 4rem);
h2: clamp(2rem, 4vw, 3rem);
h3: clamp(1.5rem, 3vw, 2rem);

/* 本文 */
body: clamp(1rem, 1.5vw, 1.125rem);
small: 0.875rem;

/* 行間 */
heading: 1.2;
body: 1.6 - 1.8;

/* 日本語は少し大きめに */
body-jp: clamp(0.9375rem, 1.5vw, 1rem);
line-height-jp: 1.8 - 2.0;
```
