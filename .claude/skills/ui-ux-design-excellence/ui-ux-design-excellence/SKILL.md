# UI/UX Design Excellence

このスキルは、高品質なUI/UXデザインを実現するための包括的なガイドを提供します。

## デザイン評価フレームワーク

UI/UXを評価・改善する際の4軸チェックリスト：

### 1. ユーザビリティ
- 目的達成までのステップ数は最小か
- 認知負荷は適切か（情報過多でないか）
- 一貫性は保たれているか
- フィードバックは適切に提供されているか

### 2. 視覚的品質
- 階層構造が明確か
- 余白は効果的に使われているか
- 色のコントラストは適切か
- タイポグラフィは読みやすいか

### 3. 独自性
- AI生成デザインの典型的パターンに陥っていないか
- ブランドの個性が反映されているか
- 記憶に残る要素があるか

### 4. インタラクション
- CTAは明確で見つけやすいか
- マイクロインタラクションは意図的か
- モバイル操作に最適化されているか

## 設計プロセスへの統合

### Step 1: 問題分析
デザイン課題を特定し、どの参照ドキュメントが関連するか判断

### Step 2: 原則の適用
`references/`内の該当ドキュメントを参照し、具体的な改善案を立案

### Step 3: AI感の排除チェック
`references/ai-aesthetics.md`で典型的なAI生成パターンを確認し、回避

### Step 4: ビジュアル調整
`references/visual-fundamentals.md`で余白・色・タイポグラフィを最適化

## 使用シナリオ別ガイド

### 新規デザイン作成時
1. `good-design.md`の原則を確認
2. `visual-fundamentals.md`で基礎を設計
3. `ai-aesthetics.md`で独自性を確保

### 既存デザインのレビュー時
1. `bad-design.md`でアンチパターンをチェック
2. 問題点を特定し改善案を作成
3. `good-design.md`の原則に照らして検証

### AI生成デザインの改善時
1. `ai-aesthetics.md`で「AI感」の特徴を確認
2. 該当するパターンを特定
3. 改善手法を適用して独自性を付与
