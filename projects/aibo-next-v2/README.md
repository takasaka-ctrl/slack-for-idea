# AIBO Next v2 - AIたまごっちコンパニオン

PDCAスコア: 8.2/10

## 概要

育てるほど賢くなる、自分だけのAIコンパニオンアプリ。
たまごっちの愛着形成 × 実用AIアシスタントの融合。

**コアバリュー**: 「育てる楽しさ × 使える賢さ。あなただけのAIパートナー」

## デザイン案

3つのデザインコンセプトを用意しました。

| 案 | スタイル | 特徴 | 閲覧 |
|----|----------|------|------|
| **A** | Retro Pixel Pop | ゲームボーイ風レトロ、温かみのあるクリーム背景 | [→ 案Aを見る](./option-a/) |
| **B** | Modern Kawaii Gaming | ソシャゲ風ポップ、パステルカラー | [→ 案Bを見る](./option-b/) |
| **C** | Neon Arcade | サイバーパンク風、ダーク背景＋ネオンカラー | [→ 案Cを見る](./option-c/) |

## MVP機能

1. **キャラクター選択** - ドット絵キャラ3種（会話型/仕事型/創作型）+ 名前設定
2. **AIチャット** - Claude API、キャラ個性はSystem Promptで制御
3. **育成システム**
   - チャット/タスク完了でXP獲得
   - レベルアップで外見変化・アニメーション解放
   - 使い方で得意分野が分岐
4. **毎日ログイン報酬** + ストリーク
5. **SNSシェアカード** - 「私のAIが Lv.XX に！」

## キャラクター仕様

- **ツール**: Aseprite（ドット絵）
- **サイズ**: 64×64〜128×128px
- **アニメーション**: 待機(4f)・話す(4f)・喜ぶ(4f)・レベルアップ(8f)

## ターゲット

- **Phase 1**: 個人（AI初心者、ゲーマー20-35歳、学生）
- **Phase 2**: ビジネスパーソン、コレクター層

## 差別化ポイント

| 競合 | 弱点 | AIBO Next v2の優位性 |
|------|------|---------------------|
| ChatGPT | 実用的だが愛着なし | 育成+愛着形成 |
| Replika | 会話AIだが実用性低 | 実用タスクもこなせる |
| Character AI | キャラはあるが育成なし | 育てるほど変化 |

## 検証すべき仮説

1. ドット絵キャラで愛着が湧くか → 7日継続率 > 40%
2. 育成×実用AIの両立 → 育成あり版が +20%継続
3. SNSシェア率 → レベルアップの5%以上がシェア
