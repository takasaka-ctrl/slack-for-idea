# AIBO Next v2 - 案A: Retro Pixel Pop

## デザインコンセプト

ゲームボーイ/ファミコン時代を彷彿とさせるレトロゲーム風デザイン。
ピクセル風ボーダー、8bit風アイコン、温かみのあるクリーム背景で
懐かしさと安心感を演出。

## 軸スコア

| 軸 | スコア |
|----|--------|
| 密度 | 3 |
| トーン | 4 |
| 時代感 | 2 |
| 彩度 | 3 |
| **合計** | **12** |

## カラーパレット

- **背景**: `#fef7e6` (クリーム)
- **プライマリ**: `#8b5a2b` (ウォームブラウン)
- **アクセント1**: `#e07020` (レトロオレンジ)
- **アクセント2**: `#40c0c0` (レトロシアン)
- **アクセント3**: `#e880a0` (レトロピンク)

## フォント

- **見出し**: Press Start 2P
- **本文**: M PLUS Rounded 1c

## 特徴

- ピクセル風ボーダーとシャドウ
- 8bit風のキャラクターデザイン
- まばたきアニメーション
- バウンスアニメーション

## ターゲット適合度

★★★★☆ (4/5)

レトロゲーム好きに刺さるデザイン。
ただし、若年層には古臭く見える可能性あり。
