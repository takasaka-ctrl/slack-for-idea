# HomeHygiene AI v2 - AI提案型掃除コーチアプリ

**PDCAスコア: 7.8/10**

## 概要

家の掃除場所を登録し、使用頻度・最終掃除日・健康リスクを元に「今日やるべき掃除」をAIが提案する掃除習慣化コーチアプリ。

**コアバリュー:** 「何を掃除すればいいか、もう悩まない」

## MVP機能

### 1. 掃除場所登録
- 部屋ごとに掃除場所を登録（キッチン、バスルーム、リビング、トイレ）
- プリセット場所 + カスタム場所追加
- 各場所に推奨掃除頻度を自動提案

### 2. 今日のおすすめ掃除TOP3
- ルールベース提案: `スコア = (使用頻度 × 2) + (最終掃除日数 × 1.5) + (健康リスク × 3)`
- スコア上位3つをカード形式で表示
- 優先度を色分けで可視化（高/中/低）

### 3. 完了記録 + ストリーク
- ワンタップで掃除完了記録
- 連続日数（ストリーク）をリング表示
- 週間進捗バーでモチベーション維持

### 4. 健康リスク可視化
- 放置日数に応じたリスクレベル表示（危険/注意/良好）
- 具体的な健康リスク説明（雑菌増殖、アレルギーなど）
- 衛生スコア（100点満点）で全体把握

### 5. プッシュ通知設定
- 毎朝の掃除提案通知（時刻カスタマイズ可能）
- 健康リスクアラート
- ストリーク応援通知
