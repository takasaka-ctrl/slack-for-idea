# PDCA評価 Draft — LaunchKit (2026-02-27 Morning)

## アイデア概要
**LaunchKit — Press Kit + Steam Page Generator for Indie Devs**

itch.io 90万ゲーム時代の発見可能性危機を解決。インディー開発者向けに、プロフェッショナルなプレスキット生成 + Steam ページ最適化を AI アシストで提供する Web アプリ。

### コアバリュー
1. **プレスキット生成**: ゲーム情報から自動でメディア向けプレスキットを生成（スクリーンショット選定、説明文、トレーラー埋め込み、fact sheet）
2. **Steam ページ最適化**: タイトル、タグ、説明文を SEO + コンバージョン最適化（A/B テスト学習データ活用）
3. **Launch Checklist**: リリース前 30 日間の TODO リスト自動生成（SNS 投稿計画、メディアリーチアウト、コミュニティ構築）

### ターゲット
- itch.io / Steam で初リリースするソロ〜小規模インディー開発者
- マーケティング経験ゼロだがゲームは完成している層
- 900K ゲームの海で埋もれたくない開発者

### 収益モデル
- Free: 1 プロジェクト、基本テンプレート
- Pro ($19/mo): 無制限プロジェクト、カスタムブランディング、Steam タグ最適化
- Launch ($49 one-time): Pro 機能 + メディアリスト 500 件 + プレスリリース配信

### エビデンス
- itch.io 90万ゲーム（2026年）→ 発見可能性が最大の課題
- Steam Next Fest 200万+ダウンロード/イベント → Steam は依然として主戦場
- Balatro（1人開発）が数百万売上 → 個人でも勝てる、ただしマーケティングが鍵
- Godot 4 参入障壁低下 → 開発者は増える一方、マーケティングノウハウは不足

## 問題点と改善案

### 問題点
1. **Check スコアが低い（17.5/25 = 7.0/10）** — 競合が多すぎる、差別化が弱い
2. **Steam 最適化の説得力** — データなしでは "AI が最適化します" は信頼されない
3. **Free ツールが多い** — Presskit() は無料で存在、なぜ $19/mo 払う？

### 改善案（Iteration 2 へ）
**Pivot: "Launch Partner" モデル → コミュニティ駆動型**

- **無料ツール + コミュニティ**にする → Freemium ではなく、基本無料で全機能開放
- **収益源を変更**: 
  - 広告なし
  - Premium: $9/mo → "Launch Partner" バッジ + メディアリスト優先アクセス + コミュニティ Slack 招待
  - データ販売: Steam 最適化データを匿名化して Unity/Epic に販売（B2B）
- **コミュニティ機能強化**: 
  - Launch Calendar（みんなのリリース予定を可視化 → 競合回避）
  - Peer Review（お互いのプレスキットをレビュー → 質向上）
  - Success Stories（誰が何万売れたか共有 → 学習効果）

このモデルなら：
- **差別化**: コミュニティは模倣困難（ネットワーク効果）
- **Distribution**: コミュニティ自体がバイラル成長の源
- **Advantage**: ユーザー増えるほどデータが貯まる → Steam 最適化精度向上 → さらにユーザー増える
