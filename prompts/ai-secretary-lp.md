# AI秘書導入サービス LP制作プロンプト

## 概要
AI秘書導入サービスのランディングページを作成する。
GitHub Pages でホストする単一HTMLファイル構成。

## リポジトリ
`takasaka-ctrl/slack-for-idea` の構成に合わせる。
- Tailwind CSS CDN（ビルドステップなし）
- Google Fonts（Space Grotesk + DM Sans + Noto Sans JP）
- 単一HTMLファイルで完結
- プロフェッショナルでモダンなデザイン

## ファイル構成
```
projects/ai-secretary/
  index.html    ← メインLP（総括 + 一人社長向けセクション）
```

## デザイン要件

### 全体方針
- **AI感のない、洗練されたプロデザイン**
- ダークモード風のヒーローセクション（信頼感）
- グラデーションアクセント（紫〜青系）
- アニメーション: スクロールフェードイン（CSS only、JSライブラリ不要）
- モバイルファースト・レスポンシブ
- 日本語ベース
- CTAボタンは目立つが上品に

### カラーパレット
- Primary: #6366f1（Indigo）
- Secondary: #8b5cf6（Violet）
- Accent: #06b6d4（Cyan）
- Dark BG: #0f172a（Slate 900）
- Text: #f8fafc / #334155

### フォント
- 見出し: Space Grotesk + Noto Sans JP
- 本文: DM Sans + Noto Sans JP

## セクション構成

### 1. ヒーロー（ダークBG）
- キャッチコピー: 「一人で全部やるの、もう終わりにしませんか？」
- サブ: 「月1.5万円のAI秘書が、24時間あなたの右腕になります。LINE・メール・カレンダーを統合管理。」
- CTAボタン: 「無料で相談する」
- 背景: グラデーション + 抽象的なドットパターン or グリッド

### 2. 課題の共感セクション
タイトル: 「こんな毎日を過ごしていませんか？」
- ✗ クライアントのLINE返信で1日が終わる
- ✗ 請求書の作成を毎回後回しにしている
- ✗ SNSの更新が3ヶ月止まっている
- ✗ 大事なメールを見落としてヒヤッとした
- ✗ スケジュール管理に使う時間がもったいない
→ 「でも、秘書を雇う余裕はない...」

### 3. 解決策セクション
タイトル: 「AIがあなたの右腕になります」
3カラムカード:
1. **LINE対応** — LINEで話しかけるだけ。予定確認、メール要約、タスク管理がチャット感覚で。
2. **メール管理** — 未読メールの要約、重要メールのアラート、返信の下書き作成を自動で。
3. **カレンダー連携** — スケジュール確認・リマインド・空き時間の提案まで。AIが24時間管理。

### 4. 会話デモセクション
タイトル: 「実際の使い方」
チャット風UIで会話例を表示:
```
あなた: 明日の予定を教えて
AI秘書: 📅 明日の予定です:
        10:00 ○○様とのZoom打ち合わせ
        14:00 企画書の締め切り
        18:00 ジム
        
        ※ Zoom打ち合わせのURLはこちらです: [リンク]

あなた: ○○さんに返信して。来週の水曜で大丈夫と伝えて
AI秘書: ✉️ 以下の返信を作成しました:
        「○○様、ご連絡ありがとうございます。
         来週の水曜日で問題ございません。
         よろしくお願いいたします。」
        
        📤 送信しますか？

あなた: 送って
AI秘書: ✅ 送信完了しました。
```

### 10. FAQ
- 「本当にLINEだけで使えるの？」→ はい。普段のLINEから話しかけるだけです。
- 「セキュリティは大丈夫？」→ データは専用サーバーで管理。第三者にデータが渡ることはありません。
- 「解約はいつでもできる？」→ はい。月額契約で、いつでも解約可能です。
- 「AI以外の対応も可能？」→ 複雑なご要望には人間のサポートが対応します。
- 「どの業種に対応していますか？」→ 基本的にどの業種でも対応可能です。

## 技術要件
- 単一HTMLファイル（index.html）

- OGPメタタグ設定
- ファビコン: emoji or インラインSVG
- パフォーマンス: 画像なし（アイコンはSVG or emoji）
