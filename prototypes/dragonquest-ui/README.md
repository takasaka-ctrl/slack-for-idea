# 🎮 ドラクエ風UI - レトロRPGスタイル PC操作インターフェース

レトロゲーム（ドラゴンクエスト）のビジュアルとUXをPC操作に適応したプロトタイプ。

---

## 🚀 クイックスタート

```bash
# ブラウザで開く
open prototypes/dragonquest-ui/index.html
```

または Canvas で表示:
```javascript
canvas.present('file:///home/node/.openclaw/workspace/prototypes/dragonquest-ui/index.html')
```

---

## 🎯 コンセプト

- **ゲーム感覚の操作**: 矢印キー + Enter でコマンド実行
- **視覚的フィードバック**: ウィンドウフレーム、カーソル、メッセージボックス
- **キーボード最適化**: マウス不要の高速ナビゲーション
- **没入感**: レトロなドット風デザインで集中力UP

---

## ⌨️ 操作方法

| キー | 動作 |
|-----|------|
| `↑` / `↓` | コマンド選択 |
| `Enter` | コマンド実行 |
| `ESC` | キャンセル/リセット |
| マウス | ホバー+クリックでも操作可 |
