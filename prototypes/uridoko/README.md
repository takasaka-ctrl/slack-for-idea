# UriDoko（ウリドコ）— どこで売るのが一番得？

フリマ・オークションの手取り額を瞬間比較するWebツール。

## 概要
商品情報（価格・重量・サイズ）を入力するだけで、メルカリ・ラクマ・Yahoo!フリマ・ヤフオク!・BOOTH・ジモティーの6プラットフォームの手取り額を自動計算・ランキング表示。

## 技術スタック
- HTML/CSS/JavaScript（バニラ、フレームワーク不要）
- Google Fonts（Noto Sans JP + Inter）
- バックエンド不要（静的サイト、データはJS内蔵）

## 機能
### MVP
1. プラットフォーム別手取り額の即座比較（6社対応）
2. 最安送料の自動選択（配送方法120+パターン）
3. カテゴリ別「売れやすさ」スコア表示
4. 手数料・送料の内訳表示
5. おすすめ梱包方法のサジェスト
6. カテゴリ別アドバイス
