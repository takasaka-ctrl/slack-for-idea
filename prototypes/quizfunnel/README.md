# QuizFunnel（クイズファネル）

診断クイズでリードを獲得する日本初の専用ツール。

## 概要
海外で実証済みの「クイズファネル」モデル（Interact: 85Mリード生成、Outgrow: 30-54% CVR）を日本市場にローカライズ。LINE連携・日本語ネイティブUI・低価格で差別化。

## エビデンス
- Interact (tryinteract.com): 85Mリード生成、40%コンバージョン率、G2 #1 Quiz Maker
- Outgrow (outgrow.co): 30-54% CVR、Fortune 500含む多数の導入実績
- 日本市場: 診断メーカー（人気だがビジネス機能なし）/ LINE MAU 9,500万人
