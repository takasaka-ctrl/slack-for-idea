# QuizFunnel（クイズファネル） — PDCA評価

## 概要
診断クイズでリードを獲得する日本初の専用ツール。海外で実証済みの「クイズファネル」モデルを日本市場にローカライズ。LINE連携・日本語ネイティブUIで差別化。

## エビデンス
- **Interact (tryinteract.com):** 2014年創業、85Mリード生成、400Mクイズ完了、40%コンバージョン率。G2で#1 Quiz Maker
- **Outgrow (outgrow.co):** 30-54%コンバージョン率、Golf Avenue事例で82,000+リード獲得
- **日本市場:** 診断メーカー（shindanmaker.com）は人気だがビジネス用途不可（リード獲得機能なし、デザインカスタマイズ不可）
- **LINE:** 日本のMAU 9,500万人。マーケティングツールとしてのLINE連携は強い差別化要因

## 技術スタック
- HTML5 / Tailwind CSS / Vanilla JS
- Google Fonts: Noto Sans JP + Inter
- LINE Messaging API（将来実装）

## UI設計
- スタイル: Clean SaaS Dashboard（Stripe/Linear風）
- カラー: #1E40AF (Primary Blue), #F59E0B (Accent Amber), #F9FAFB (Background)
- フォント: Noto Sans JP (本文), Inter (数字・英語)
- 参考: Typeform, Interact, Linear

## 機能仕様
### MVP（2週間以内）
1. クイズ作成フォームビルダー（ドラッグ不要、ステップ型）
2. 結果ページカスタマイズ（テンプレート3種）
3. リード収集（メールアドレス + LINE友だち追加ボタン）
4. 分析ダッシュボード（回答数、完了率、リード数）
5. 埋め込みコード生成（iframe/スクリプト）

### Phase 2（将来）
- LINE Messaging API連携（自動セグメント配信）
- テンプレートマーケットプレイス
- A/Bテスト機能
- Webhook連携（Zapier等）
